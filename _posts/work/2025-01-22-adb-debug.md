## Adb调试





###  adb抓设备日志
> 开启adb,debug模式下

* 清理 20m空间

>  adb logcat -G 20M
>  

* 清除之前的缓存

  > adb logcat -c
  > 
``` shell
# 查看设备
adb devices

# 1. 清理 20m空间

adb logcat -G 20M

# 2.清除之前的缓存
adb logcat -c

# 3.写出日志到文件
adb logcat > ./log/log.txt
```

# PC工具
